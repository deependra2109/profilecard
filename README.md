Profile Card UI

A modern Profile Card UI built using HTML and CSS.
This project displays a user profile with image, name, role, social media icons, and interactive buttons in a clean and responsive layout.

---



## 📌 Features

* Responsive profile card design
* Circular profile image styling
* Social media icons integration
* Clean and modern UI layout
* Built using **Flexbox for alignment**
* Simple and beginner-friendly code structure

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Boxicons

---
